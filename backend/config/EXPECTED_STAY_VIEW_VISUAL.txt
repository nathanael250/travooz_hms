BOOKING BAR DETAILS:
═══════════════════════════════════════════════════════════════════════════════

Width Calculation:
- Each day column = 128px
- Booking bar width = nights × 128px
- Example: 3-night stay = 384px wide

Position:
- Starts at check-in date column
- Spans across multiple date columns
- Positioned absolutely within the grid cell

Text Display:
- Guest name shown in the bar
- Truncated with ellipsis if too long
- Full name visible on hover (title attribute)
